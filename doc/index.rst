******************************************************************************
geodas - Geospatial Data Analysis in Python
******************************************************************************

|geodas| is a Python_ library aimed at the analysis of geospatial data. Its 
main purpose is the handling of large 4-dimensional datasets (time, altitude,
longitude, latitude), and it is geared towards easy interactive use, i.e.
in combination with ipython_.
