******************************************************************************
Source code documentation
******************************************************************************

|geodas| is developed with the `literate programming`_ paradigm in mind. We
use pylit_ to generate ``.rst`` files from the Python source code and
include these in the Sphinx documentation. This source code documentation
is mainly meant for developers; users are referred to the :ref:`api`.
